🔗 API エンドポイント

システム初期化

POST /api/setup/initialize
Authorization: Bearer [CRON_SECRET]

Webhook受信

POST /api/webhook/calendar?token=[WEBHOOK_SECRET]

Webhook更新（Cron）

POST /api/cron/refresh-webhooks
Authorization: Bearer [CRON_SECRET]
スケジュール: 毎日午前2時
